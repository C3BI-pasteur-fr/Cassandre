Interface configuration

Cassandre uses the first line of your files (the column names) and the content of the first column (the row names) to store your data and allow you to explore it properly.

Here you can put a description about what rows and columns are in your data: genes identifiers, experiment names, strain names..

The interface will display those informations everywhere it is needed. You can change those parameters at any time.
